# Regional Sales Comparison Dashboard | Tableau

## Objective
Visualize and compare sales performance between selected regions using an interactive dashboard to support strategic decision-making.

## Tools Used
- Tableau

## Project Overview
This project analyzes regional sales performance through interactive visualizations. Users can compare two regions side-by-side and evaluate trends over time.

## Process
- Built interactive Tableau dashboard with time-based visualizations
- Added filters for region and time selection
- Enabled side-by-side comparison for performance analysis

## Results
Created a dynamic dashboard providing clear regional insights that help stakeholders evaluate performance and optimize sales strategy.

## Key Features
- Interactive filters
- Comparative regional analysis
- Time-series trend visualization
